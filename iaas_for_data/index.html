BigQuery Analytics

Production data loaded and ready for analysis

BigQuery table with normalized FX rates data‚Äîpartitioned by ingestion date, ready for SQL analytics and visualization tools like Looker Studio or Tableau.

-- Example query: Latest EUR to USD exchange rate
SELECT 
  date,
  currency,
  rate,
  ingestion_date
FROM `analytics.raw_fx_rates`
WHERE currency = 'USD'
  AND ingestion_date = (SELECT MAX(ingestion_date) FROM `analytics.raw_fx_rates`)
ORDER BY date DESC
LIMIT 1;

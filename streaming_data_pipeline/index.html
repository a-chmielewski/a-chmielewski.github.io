Data Schema

Processed trade events with the following structure

{
  "timestamp": "2025-11-03T12:30:45.123Z",
  "symbol": "BTCUSDT",
  "price": 107572.10,
  "quantity": 0.045,
  "is_buyer_maker": false,
  "trade_id": 1234567890,
  "trade_time_ms": 1730638245123
}

timestamp

ISO 8601 formatted trade execution time

symbol

Trading pair (e.g., BTCUSDT, ETHUSDT)

price

Trade execution price (double precision)

quantity

Trade volume in base currency

is_buyer_maker

Whether buyer was market maker (boolean)

trade_id

Unique aggregate trade identifier
